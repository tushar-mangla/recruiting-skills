# Sourcing Searches: Software Project Manager

## Step 1 — Exact Match

**LinkedIn free search:**
```
"Software Project Manager" AND ("Agile" OR "Scrum") AND ("SaaS" OR "B2B") AND Prague
```

**LinkedIn Recruiter:**
```
Title: "Software Project Manager" OR "IT Project Manager"
Skills: Agile, Scrum, Stakeholder Management
Location: Czech Republic (or EU remote)
Industry: Computer Software, Internet
```

**Google X-ray:**
```
site:linkedin.com/in/ "software project manager" "Agile" "SaaS" "Prague"
```

## Step 2 — Broaden Titles

**LinkedIn:**
```
("Software Project Manager" OR "IT Project Manager" OR "Technical Project Manager" OR "Digital Project Manager") AND ("Agile" OR "Scrum") AND ("SaaS" OR "B2B" OR "tech")
```

**Google X-ray:**
```
site:linkedin.com/in/ ("project manager" OR "program manager") ("software" OR "tech" OR "IT") "Agile" "Czech"
```

```
("Delivery Manager" OR "Scrum Master" OR "Agile Coach" OR "Release Manager" OR "Technical Program Manager") AND "software" AND ("Prague" OR "Czech" OR "remote")
```
